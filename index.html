Project

Short Description

This project offers users the ability to generate QR codes leading to their own or other freely chosen websites in 4 different colors. Once the QR code is created, it can be freely downloaded in PNG format. Additionally, the URL field is validated using regex patterns, reducing the chances of errors and ensuring the creation of QR codes that lead only to valid URLs.

Key Features

  • Generate QR Codes in 4 Different Colors
  • Download QR Codes in PNG Format
  • URL Validation with Regex Patterns
  • Utilize ng Directives for Element Disappearances
  • Responsive Design

Project

Short Description

This PHP project consists of two parts: the first part handles gallery creation and the second part manages image uploads to existing galleries. In the gallery creation section, it checks if a gallery name is provided, creates a new directory for the gallery if it doesn't already exist, and provides feedback on the success of the creation. In the image upload section, it allows users to select a gallery, upload multiple images, and validates that the uploaded files are in suitable formats (JPEG, PNG, or WebP).

Key features

  • Efficient Gallery Creation
  • Informative Feedback
  • User-Friendly Image Upload
  • Responsive Design
